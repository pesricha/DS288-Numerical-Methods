def display_side_by_side(dfs:list, captions:list, tablespacing=5):
    """Display tables side by side to save vertical space
    Input:
        dfs: list of pandas.DataFrame
        captions: list of table captions
        ## This function is taken from https://stackoverflow.com/questions/38783027/.
    """
    output = ""
    for (caption, df) in zip(captions, dfs):
        output += df.style.set_table_attributes("style='display:inline'").set_caption(caption)._repr_html_()
        output += tablespacing * "\xa0"
    display(HTML(output))

Forward and Backward absolute errors against n

From the above graph, we can make the following observations:

  1. The error growth for x = 1 for both forward and backward pass is exponential.
    1. The error in forward pass grew from order of $10^{-6}$ to order of $10^{2}$
    2. The error in backward pass grew from order of $10^{-15}$ to $10^{6}$
  2. The error growth for x = 5 for forward pass seems to be exponential at around from n = {6 to 10}. For backward pass, the error seems to be exponential till n = {10 .. 4} and then juggling randomly (not exponential).
    1. The error in forward pass grew from order of $10^{-6}$ to order of $10^{-4}$
    2. The error in backward pass grew from order of $10^{-9}$ to $10^{-6}$
  3. The error growth for x = 50 is juggling randomly (not exponential) for both forward pass and backward pass.
    1. The error in forward pass maintained in order of $10^{-7}$
    2. The error in backward pass maintained in order of $10^{-6}$

We know that to find the class of error growth, we have to write the iterative scheme. $$J_{n}(x) = \frac{2(n-1)}{x}J_{n+1}(x) - J_{n+2}(x)$$

Now let us assume that the $\frac{2(n-1)}{x}$ is a constant represented by $\beta$. Our scheme then becomes: $$J_{n}(x) = \beta J_{n-1}(x) - J_{n-2}(x)$$

Where $\beta$ varies with the input x. We can calculate the maximum and minimum values of $\beta$ to get some intuition about the errors.

As discussed in class, we cannot represent the numbers to their exact precision in computer. Let $\hat{J}_{n}(x)$ represent the representation we can have in our computers. $$\hat{J}_{n}(x) = \beta \hat{J}_{n-1}(x) - \hat{J}_{n-2}(x)$$

Subtracting equation (2) from (1) we get: $$e_{n}(x) = \beta e_{n-1}(x) - e_{n-2}(x)$$

Now, let's assume the error class is exponential

$$e_n \propto k^n \cdot e_0 \\ \implies k^n = \beta \cdot k^{n-1} - k^{n-2} \\ \implies k^n - \beta \cdot k^{n-1} + k^{n-2} = 0 \\ \implies k^{n - 2} \left( k^2 - \beta k + 1 \right) = 0 \\ k = \frac{\beta \pm \sqrt{\beta^2 - 4}}{2}$$

Since Beta is positive the roots of the equation will be real only if $ \beta \geq 2$, which is also the condition for which the error growth will be exponential. We will calculate the values of $\beta$ that we will get for both forward and backward pass.
